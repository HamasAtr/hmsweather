
# HmsWeather

This is a html,css,js weather app made with RapidApi

## Authors

- [@HamasAtr](https://www.github.com/HamasAtr)


## Run Locally

Clone the project

```bash
  git clone https://github.com/HamasAtr/hmsweather
```



Install dependencies

```bash
  liveserver install
```

Start the server

```bash
  open with liveserver
```
